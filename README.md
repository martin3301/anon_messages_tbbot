# 🕵️ anon_messages_tg

Телеграм-бот для анонимной отправки сообщений и комментариев в канал. Пользователь взаимодействует с ботом в личке, а сообщения публикуются в канале от имени случайно сгенерированного псевдонима.

---

## ✨ Возможности

- **Анонимный текст** — отправка текстового сообщения в канал
- **Текст + медиа** — отправка альбома (до 10 фото/видео) с подписью
- **Анонимный комментарий** — ответ на конкретный пост в канале через группу комментариев
- **Проверка подписки** — бот пропускает только подписчиков канала
- **Псевдонимы** — каждому пользователю назначается случайное имя, которое обновляется каждые ~6 часов (4 периода в сутки)

## ⚙️ Установка и запуск

### 1. Клонировать репозиторий

```bash
git clone <repo-url>
cd anon_messages_tg
```

### 2. Установить зависимости

```bash
pip install -r requirements.txt
```

### 3. Настроить `config.py`

```python
TOKEN = 'токен_вашего_бота'        # Токен от @BotFather
mainChannel = 'id_канала'          # ID канала для публикаций (например, -1001234567890)
commGroup = 'id_группы_комментариев'  # ID группы комментариев канала
```

### 4. Добавить бота в канал и группу комментариев

- Бот должен быть **администратором** в канале (права на публикацию)
- Бот должен быть **администратором** в группе комментариев (права на отправку сообщений)

### 5. Запустить бота

```bash
python run.py
```

## 🗄️ База данных

Используется SQLite (`database.db`), создаётся автоматически при первом запуске.

**Таблица `users`** — хранит данные пользователей и текущий псевдоним.

**Таблица `messages`** — связывает ID поста в канале с ID сообщения в группе комментариев (нужно для анонимных ответов).

---

## 🔄 Как работает анонимность

Каждый пользователь получает псевдоним вида `FirstName LastName` (например, `Tyler Brooks`). Псевдоним привязан к временному периоду (~6 часов) — после его смены имя обновляется. В комментариях дополнительно генерируется уникальный короткий код вида `a12345`.

## 📝 Использование бота

1. Пользователь запускает бота командой `/start`
2. Бот проверяет подписку на канал
3. Выбирает тип сообщения:
   - **Только текст** — пишет текст, он уходит в канал
   - **Не только текст** — сначала отправляет фото/видео, затем текст с тегом `#end`
   - **Анонимный коммент** — вводит ID поста из канала, затем отправляет комментарий (для выхода пишет `#end`)
